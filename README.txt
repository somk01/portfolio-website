Somya Kateja — Portfolio (standalone static site)

How to run:
1. Open this folder in VS Code.
2. Install the "Live Server" extension (Ritwick Dey).
3. Right-click index.html → "Open with Live Server".

Or just double-click index.html to open in a browser.

No build step, no Node, no npm. Pure HTML + CSS + Google Fonts.

Files:
  index.html              Home page
  styles.css              All styles
  work/*.html             4 campaign concept pages
  stories/*.html          Short-form story (14 episodes)
